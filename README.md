# 🛒 Proyecto: SportyStyle - Tienda Virtual

## 📌 Descripción
SportyStyle es una tienda virtual desarrollada con HTML, CSS y JavaScript.  
Permite a los usuarios visualizar productos deportivos, agregarlos a un carrito de compras y realizar una compra ingresando sus datos personales.

---

## 🎯 Objetivos del Proyecto
- Simular el funcionamiento de una tienda online
- Manipular el DOM con JavaScript
- Usar sessionStorage
- Validar formularios

---

## ⚙️ Funcionalidades

### 🧍 Inicio de sesión
Se simula con un prompt que guarda el nombre en sessionStorage y muestra un mensaje de bienvenida.

---

### 🛍️ Productos
- Camiseta
- Pantalón
- Accesorio

Se pueden agregar al carrito.

---

### 🛒 Carrito
- Guarda productos
- Calcula total
- Usa sessionStorage

---

### 🧾 Compra
Formulario con:
- Nombre
- Dirección
- Correo
- Teléfono

Validaciones:
- correo con "@"
- teléfono numérico

---

### 💬 Confirmación
Muestra nombre, dirección y total.

---

## 🔐 Auth0 (Intento)
Se intentó implementar Auth0 para autenticación real, pero hubo problemas de configuración y redirección.

Por eso se utilizó sessionStorage como solución funcional para cumplir los objetivos.

---

## 📊 Conclusión
Proyecto funcional con carrito, validaciones y sesión simulada.
